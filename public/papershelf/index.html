Current Reading

Research Papers

Designing Data-Intensive Applications
Martin Kleppmann
O'Reilly Media, 2017

An excellent deep dive into the fundamentals of distributed systems, data storage, and processing. Essential reading for understanding how to build scalable applications.

The Google File System
Sanjay Ghemawat, Howard Gobioff, Shun-Tak Leung
SOSP 2003

Foundational paper on distributed file systems that influenced many modern distributed storage solutions.

MapReduce: Simplified Data Processing on Large Clusters
Jeffrey Dean, Sanjay Ghemawat
OSDI 2004

Revolutionary approach to processing large datasets that spawned numerous big data frameworks.
